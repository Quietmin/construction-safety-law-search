# PRD — 건설안전 법령 검색기

> **제품명:** 건설안전 법령 검색기 (산업안전보건법 검색 웹앱)
> **문서 버전:** v1.0
> **작성일:** 2026-06-08
> **상태:** v1.0 구현 완료 (`index.html` 배포 가능)

---

## 1. 개요 (Summary)

건설공사 안전관리 담당자가 **현장 용어로 검색하면 관련 산업안전보건 법령 조문과 현장에서 확인할 조치 항목(체크리스트)을 즉시 보여주는** 단일 HTML 웹앱.

서버·인터넷 연결 없이 **HTML 파일 하나를 더블클릭**하면 동작하며, 법령 데이터(3개 법령·647개 조문)가 파일에 내장되어 있다.

---

## 2. 배경 및 문제 (Background & Problem)

- 산업안전보건 법령은 **본법 + 시행령 + 산업안전보건기준에 관한 규칙**으로 흩어져 있고, 조문 수가 많아(647개) 필요한 기준을 찾기 어렵다.
- 현장 담당자는 법률 용어가 아닌 **현장 용어(비계, 추락, 굴착 등)**로 사고한다.
- "이 작업을 어떤 기준으로, 무엇을 조치하며 해야 하는가"를 빠르게 확인할 수단이 부족하다.
- 공공 법령 API는 인증키 발급과 CORS 제약으로 순수 HTML 앱에서 직접 호출이 어렵다 → **법령 원문을 파일에 내장**하는 방식 채택.

---

## 3. 목표 (Goals) / 비목표 (Non-Goals)

### 목표
- 키워드 한 번으로 관련 **법 조문 원문**을 정확히 찾는다.
- 해당 조문에서 **현장 조치/체크리스트를 자동 생성**해 실무에 바로 활용한다.
- **오프라인·무설치**로 어떤 PC에서나 즉시 사용한다.

### 비목표 (v1 범위 밖)
- 법령 실시간 자동 업데이트(API 연동)
- 법률 자문·유권해석 제공 (체크리스트는 참고용, 법적 효력 없음)
- 사용자 계정·서버 저장·다중 사용자 협업

---

## 4. 대상 사용자 (Target Users)

| 사용자 | 사용 맥락 |
|--------|-----------|
| 건설현장 안전관리자 | 사무실 PC에서 작업 전 적용 기준 확인 |
| 현장소장 / 공사 관리자 | 적법 시공 여부 점검 |
| 안전 교육·점검 담당 | 체크리스트 기반 현장 점검 |

- **주 사용 환경:** PC (넓은 2단 레이아웃). 모바일/태블릿도 반응형 지원.

## 5. 핵심 기능 (Features)

### F1. 키워드 검색
- 검색창에 입력한 단어가 **포함된 모든 조문**을 검색.
- 공백으로 여러 단어 입력 시 AND 조건.
- **관련도 정렬**: 조문 제목·번호에 단어가 있으면 가중치(+100), 본문 등장 횟수 합산.
- 결과에 **검색어 하이라이트** 및 2줄 미리보기.

### F2. 법령별 필터
- `전체 / 법 / 시행령 / 안전보건규칙` 칩으로 범위 제한.

### F3. 빠른 검색(자주 찾는 항목)
- 비계, 추락 방지, 굴착, 거푸집 동바리, 안전난간, 밀폐공간, 전기, 크레인, 안전모, 도급, 안전보건교육, 중대재해.

### F4. 조문 상세 — 법 조문 원문
- 선택 조문의 **원문을 가공 없이 그대로** 표시(법적 근거 보존).
- 출처 법령 배지 + 장(章) 표시 + 검색어 하이라이트.

### F5. 현장 조치 체크리스트 (자동 생성)
- 조문 본문에서 의무·조치 문장을 추출해 **체크박스 목록**으로 변환.
  - 추출 대상: `~할 것`, `~하여야 한다`, `~해야 한다`, `~해서는 아니 된다`, `~받아야 한다` 등.
  - 제외: 적용 예외 문구(`~할 수 없다`, `예외로 한다`).
- 항(①②)·호(1.2.)·목(가.나.) 단위로 분해해 항목화. 체크 시 완료 표시(취소선).
- **면책 안내 문구 상시 노출**: "참고용이며 법적 효력 없음, 원문·전문가 확인 필요".

## 6. 데이터 (Data)

| 항목 | 내용 |
|------|------|
| 원천 | 국가법령정보센터 공개 법령 (사용자 제공 `.doc`) |
| 포함 법령 | 산업안전보건법 [시행 2026.6.1.] / 시행령 [2026.3.24.] / 산업안전보건기준에 관한 규칙 [2026.3.2.] |
| 조문 수 | 법 193 + 시행령 125 + 규칙 329 = **647** |
| 가공 | RTF 파싱 → 이미지/헤더 노이즈 제거 → 조문 단위 구조화(JSON) → HTML 내장 |
| 데이터 구조 | `{ law, lawKey, article, jo, sub, chapter, title, body }` |

**미포함(현재):** 별표(별표 1~30 등) — 구체 수치 기준 일부가 별표에 존재.

## 8. 비기능 요구사항 (Non-Functional)

- **무설치/오프라인:** 단일 `index.html`, 외부 의존성 없음. `file://`로 직접 실행 가능.
- **성능:** 647개 조문 클라이언트 검색, 입력 즉시(수십 ms) 응답.
- **호환성:** 최신 Chrome/Edge 기준. 한글 폰트(맑은 고딕) 사용.
- **유지보수:** 법령 개정 시 `.doc` 재추출 → `build_data.py`로 데이터 재생성 → 템플릿 주입.

---

## 9. 제약 및 면책 (Constraints & Disclaimer)

- 본 도구는 **현장 참고용**이며 **법적 효력이 없다.**
- 체크리스트는 조문에서 자동 추출한 것으로, **누락·오추출 가능성**이 있다.
- 실제 적용 시 **법령 원문과 안전관리자 등 전문가의 확인**이 반드시 필요하다.
- 법령은 개정되므로 **시행일자 기준** 확인이 필요하다.
