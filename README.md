# 건설안전 법령 검색기 (LEX-CON Safety Law)

현장 용어로 검색하면 관련 **산업안전보건 법령 조문**과 **현장 조치 체크리스트**를 즉시 보여주는 단일 HTML 웹앱입니다. 서버·인터넷 연결 없이 `index.html` 하나를 더블클릭하면 동작하며, 법령 데이터가 파일에 내장되어 있습니다.

## 주요 기능

- **스마트 키워드 검색** — 띄어쓰기·합성어를 자동 보정하고(`강관비계 설치기준` 등), 현장용어 동의어(낙하→추락 등)를 매핑해 관련 조문을 관련도순으로 표시
- **법령별 필터** — 17개 법령/고시 범위 제한
- **조문 전문(원문) 표시** — 선택 조문을 가공 없이 그대로
- **현장 조치 체크리스트 자동 생성** — 의무 문장을 추출해 체크박스로 변환, 진행률 표시
- **별표 검색** — 수치 기준이 담긴 별표/별지 텍스트 포함
- **다중 화면** — Dashboard / Laws / Safety Check / Reports / Archive / Help

## 수록 법령 (총 17종 · 2,400+ 조문/별표)

| 분류 | 법령 |
|------|------|
| 산업안전보건 | 산업안전보건법 · 시행령 · 시행규칙 · 안전보건기준에 관한 규칙 |
| 중대재해 | 중대재해처벌법 · 시행령 |
| 건설기술 | 건설기술 진흥법 · 시행령 · 시행규칙 |
| 시설물 | 시설물안전법 · 시행령 · 시행규칙 |
| 기타 | 건설산업기본법 · 유해위험작업 취업제한규칙 |
| 고시 | 건설업 안전보건관리비 계상·사용기준 · 위험성평가 지침 · 안전보건교육규정 |

## 사용 방법

`index.html` 을 브라우저로 열면 됩니다. (오프라인·무설치)

## 데이터 재생성

```bash
# 1) (선택) 최신 법령을 국가법령정보센터 Open-API에서 수집
python fetch_all.py <본인_OC_키>     # open.law.go.kr 에서 무료 발급

# 2) .doc(RTF) + fetched_laws.json 을 구조화하고 index.html 생성
python build_data.py
```

| 파일 | 역할 |
|------|------|
| `index.html` | **최종 배포물** (데이터 내장, 더블클릭 실행) |
| `template.html` | UI·로직 템플릿 |
| `extract_rtf.py` | RTF(.doc) → 텍스트 추출 |
| `fetch_all.py` | Open-API로 법령·고시·별표 수집 → `fetched_laws.json` |
| `build_data.py` | 구조화 + 템플릿 주입 → `index.html` |
| `*.doc` | 사용자 제공 법령 원문(RTF) |

## 면책

본 도구는 **현장 참고용**이며 **법적 효력이 없습니다.** 체크리스트는 조문에서 자동 추출한 것으로 누락·오추출 가능성이 있으며, 일부 표·이미지형 별표는 텍스트가 거칠 수 있습니다. 실제 적용 시 **법령 원문과 안전관리자 등 전문가의 확인**이 필요합니다. 법령 원문은 [국가법령정보센터](https://www.law.go.kr) 공개자료입니다(저작권법 §7, 보호 대상 아님).
